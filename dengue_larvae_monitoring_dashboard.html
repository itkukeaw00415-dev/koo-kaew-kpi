ตั้งค่าการเชื่อมต่อฐานข้อมูล

ระบบพัฒนาขึ้นมารองรับระบบ Cloud ผ่าน Google Sheets + Google Apps Script Web App โดยหากยังไม่มีระบบหลังบ้าน สามารถรันโหมดจำลองสถานการณ์ไปก่อนได้ครับ

คัดลอก Web App URL จากส่วนการปรับใช้ (Deployment) บน GAS ของคุณมาวาง

ขั้นตอนการทำฐานข้อมูลคลาวด์ของตนเอง:

  1. สร้าง Google Sheets เปล่าขึ้นมา
  2. ไปที่ Extensions -> Apps Script
  3. นำโค้ดในไฟล์ gas_code.js ไปวางแทนที่แล้วกด Save
  4. กด Deploy -> New deployment -> เลือก Web App
  5. ตั้งผู้เข้าถึงเป็น "Anyone" แล้วนำลิงก์มาวางในช่องด้านบนนี้
